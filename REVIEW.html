What this does: Makes the prominent "join the org" message (from PR #627) configurable via a new OrgInviteConfig struct nested inside the existing Trigger config.

New config fields: disabled (skip prominent message), merged_pr_threshold (customize PR count, default 3), message (custom template with {join_org_url} placeholder).

Files changed: config.go, pull-request.go, pull-request_test.go, trigger.go
