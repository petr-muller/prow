Remaining findings

Worth discussing Fragile DOM-walking hide script

pkg/spyglass/lenses/markdown/markdown.go:131-138

When no documents match the file filter, the lens returns a <script> that hides its container by walking up the DOM tree:

const container = window.frameElement.parentElement.parentElement;
if (container) {
    container.style.display = 'none';
}

This relies on the specific DOM nesting of the Spyglass layout. If the layout changes, this will break silently (the lens panel stays visible but empty, or worse, hides a parent it shouldn't). Worth asking if there's a standard Spyglass mechanism for a lens to signal "I have nothing to show."
